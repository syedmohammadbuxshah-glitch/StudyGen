# StudyGen — Live API Frontend Practice

## Assignment completed

This project includes a live frontend API experience at:

`/api-lab.html`

### Features
- Fetches live country data from the public REST Countries API
- Visible loading state while data is being fetched
- Successful data state with country cards
- Search/filter by country name, region, or capital
- Friendly error state when the API request fails
- A **Test error state** button for recording a clear failure-state demo
- Reload button to retry the API request

### Demo flow for video
1. Open the API Lab page and show the loading state.
2. Wait for the country cards to load.
3. Search for a country, region, or capital.
4. Click **Test error state** and show the friendly error UI.
5. Click **Reload data** to recover successfully.

The main StudyGen application remains available through the existing app entry point.
